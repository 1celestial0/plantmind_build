## 2. PROJECT: PLANTMIND

### What it is
An **agentic Engineering Intelligence layer** that converts industrial sensor streams into ranked, explainable corrective actions — and *proves* each decision by flipping a failing asset from RED to GREEN using counterfactual simulation.

**Engine name:** Götze Decision Engine (GDE)
**Tagline:** *Predict the failure. Decide the fix. Prove it.*
**Dataset:** NASA C-MAPSS (21 sensors, run-to-failure, turbofan degradation)
**Stack:** Python · scikit-learn · Streamlit · Plotly · LLM API · Databricks CE

### The 5-Layer Architecture
```
LAYER 1 · DATA         → NASA C-MAPSS → clean → 30-cycle sliding window
LAYER 2 · PREDICTION   → Random Forest → RUL score → RED flag if RUL < 30
LAYER 3 · DIAGNOSIS    → LLM agent → causal graph → root cause → action menu
LAYER 4 · COUNTERFACTUAL → Surrogate twin per action → Götze Score
LAYER 5 · PROVE+LEARN  → RED→GREEN replay → dashboard → weight recalibration
```

### The Core IP (Götze Score Formula)
```
G = w1·ΔHealth + w2·(1−NormCost) + w3·(1−NormTime) + w4·Safety
Weights: health=0.40, cost=0.25, time=0.20, safety=0.15
DESIGN RULE: AI does uncertain work. Deterministic math makes the final decision.
```
