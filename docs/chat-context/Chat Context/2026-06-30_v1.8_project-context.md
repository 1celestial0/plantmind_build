## What is PlantMind x GotzeEngine
A 5-agent predictive maintenance pipeline for LTTS industrial plants:
- DataSentinel -> AssetHealthOracle -> GotzeEngine -> RootCauseAnalyst -> ExecutiveSummarizer
- Physics-informed Weibull health model: H(t) = 100*exp(-lambda*S*t^beta)
- IIS formula (LOCKED_STATE 2): 0.35*delta_P_failure + 0.25*delta_DowntimeCost + 0.20*Feasibility + 0.15*HistoricalSuccess - 0.05*SafetyRiskDelta
- Human-in-the-loop approval gate before any maintenance action executes
- Demo narrative: Jamnagar Refinery pump_07, health 18.8/100, GotzeEngine recommends predictive shutdown (IIS 0.868), saving $470k vs unplanned failure
